Mijn HTML/CSS progress








Reflectie

Ik heb met dit keuzevak de basiselementen geleerd van HTML/CSS. Ik begon met het opzetten van mijn eigen Github-account en het gebruiken van Visual Studio Code. Ik heb met het vak geleerd hoe ik een pagina kan indelen met header, main en footer. In deze onderdelen heb ik verschillende tags leren gebruiken, zoals: ul/ol, section, article en p.

Om deze onderdelen te kunnen opmaken, heb ik geleerd hoe ik in CSS kan refereren naar een class, id of ander onderdeel in HTML. Eerst koppelde ik CSS aan HTML. Daarna begon ik met de opmaak. Dit deed ik bijvoorbeeld door de (background)color van teksten en vormen te veranderen. Ook leerde ik hoe ik vormen, tekst en bijvoorbeeld afbeeldingen kon uitlijnen naar een bepaalde kant.

Uiteindelijk heb ik verschillende opdrachten kunnen maken, zoals een simpele website met daarop een recept van een vegetarische burger. Actiepunten

Ik zou graag meer willen leren over het opmaken van HTML. Er zijn enorm veel tags in CSS die voor een gave look kunnen zorgen op een website. Ik zou ook graag meer willen leren over het opmaken/invoeren van afbeeldingen en navigatie/sidebars. Deze punten zijn nog niet echt in het keuzevak voorgekomen, maar is wel handig om te weten bij het maken van een website.

  • Verder leren hoe ik vormen/teksten/afbeeldingen kan opmaken met CSS
  • Leren hoe ik sidebars/andere navigatiebalken moet maken in HTML

Toekomstplannen

Ik zou graag voor mijn portfolio later een pagina willen maken die ik kan delen als mensen op mijn naam zoeken. Dit zou handig zijn als ik bijvoorbeeld aan het zoeken ben naar een stageplek. Op die pagina zou ik graag alle belangrijke projecten willen vermelden die ik zelfstandig heb geleid.
